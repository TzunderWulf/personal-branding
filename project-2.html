Roamer

Roamer was a small webapplication built with pure HTML and CSS, to see what you can do with the current weather. The design is also mobile-friendly.

After filling in the necessary data, the application will predict the weather and give some sights in Rotterdam you could check out. As an example if it rains, it will give indoor activities.

The project was built with pure HTML, CSS and JS. For the machine learning aspect the ML5.js library was used, specifically the neural network.
