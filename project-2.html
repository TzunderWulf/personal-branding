RESTful Webservice

A back-end webservice to store, add, edit an delete characters. Built for the course programming-6 at the Rotterdam University of Applied Sciences. Also features a front-end to interact with the webservice, built with React as a seperate part of the course.

The image on the left is of the front-end, with which you can interact with the webservice. The project sadly is no longer online, as it was hosted on the student servers.

The project was built with the MERN stack: MongoDB, Express, React and Node.
