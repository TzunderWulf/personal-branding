Genshin Archive

Project for the class PRG05, using the Laravel framework. Built a archive for the open-world RPG game: Genshin Impact as a school project.

You can edit, remove or disable posts/characters you've created, admins can edit, remove and disable every character. As an user you can favourite characters, which allows you to be able to add characters. You can search with a query or tags.

The project was built with the Laravel framework.
